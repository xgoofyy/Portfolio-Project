# 💻 My Portfolio Site

This is the repo for my personal portfolio website. This site showcases my journey as a student, my engineering projects, and my various involvements.

**Live Link:** [My Site](www.arnoldzamora.me)

## 🚀 About the Project

This site was built to be a fast, responsive, and visually engaging hub for my professional work. It features a custom hero animation, a responsive project showcase, and a functional contact portal.

### **Core Features:**

* **Dynamic Animations:** Custom CSS keyframe animations for the "rolling" hero text.
* **Responsive Project Carousel:** A scroll-snapping project gallery optimized for all device widths.
* **Contact Integration:** A fully functional contact form powered by Web3Forms.

## 🛠️ Tech Stack

### **Frontend**

* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom Flexbox layouts, CSS Variables, and advanced Keyframe animations.
* **FontAwesome:** Scalable vector icons for social links.

### **Tools & Libraries**

* **Web3Forms:** For serverless contact form handling.
* **DevIcons:** For technical stack visualization in project cards.
* **Google Fonts:** Utilizing the "Poppins" typeface family.

## 📂 Project Highlights

The site features several of my key engineering projects, including:

* **Hoopmaster:** A 3D-printed electromechanical toy (ID3EA Course).
* **Python to Arduino Tracking:** Low-latency hardware-software synchronization.
* **Facial Recognition Actuators:** Real-time computer vision using OpenCV.

And more that can be found on my GitHub, LinkedIn, or resume

## 📱 Responsive Design

The site is built with a **Desktop-First** approach, utilizing specific breakpoints to ensure a "mobile-app" feel on smaller devices:

* **1200px+:** Full-screen hero with side-by-side content.
* **1024px:** Fluid scaling of images and text.
* **510px:** Stacked vertical layouts for optimal mobile readability.
